# **App Name**: Craftify AI: Flutter App Builder

## Core Features:

- AI-Powered App Generation: Generate Flutter app code from a text description using the OpenAI GPT API, tool (Genkit).
- Authentication: User authentication using Firebase Authentication (Google & email login).
- Dashboard: Centralized dashboard to create new apps, access recent projects, and manage account.
- Project Management: Create and store Flutter projects including descriptions and themes.
- Code Preview: Web-based and Phone-based, live preview of the generated Flutter app's UI and navigation using JSON schema.
- Code Export: Download generated Flutter code as a ZIP file with setup instructions, served from the server.
- User profile and Settings: Settings to view profile, API stats and notification settings

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a trustworthy, innovative feel.
- Background color: Light gray (#F5F5F5), very desaturated, providing a clean backdrop.
- Accent color: Purple (#7E57C2), analogous to blue, yet distinctly different, providing contrast for interactive elements.
- Headline font: 'Space Grotesk' sans-serif, used for its computerized, techy, scientific feel; body text: 'Inter' sans-serif, to add clear readability to blocks of text
- Code font: 'Source Code Pro' for displaying code snippets.
- Modern and minimalist icons to represent different app features and functionalities.
- Subtle animations and transitions to enhance user experience during app generation and navigation.
